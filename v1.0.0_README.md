#### 🎯 新窗口处理（关键突破）
**问题**: 启信宝点击公司名会在新标签页打开
**解决方案**:
- 检测标签页数量变化
- 自动切换到最新标签页
- 在正确的页面上提取数据

#### ⏱️ 多层次等待机制
**问题**: 搜索结果加载慢，容易过早点击
**解决方案**:
- 等待 3 秒让列表渲染
- 等待"查询结果"指示器出现
- 等待网络空闲状态
- 额外等待 2 秒确保动态内容加载

#### 🔍 智能选择器系统
**问题**: 页面结构可能变化
**解决方案**:
- 每个字段 9+ 个备选选择器
- 按优先级依次尝试
- 记录使用的具体选择器
- 支持模糊匹配（如 `a[href*="/company/"]`）

## 💡 使用技巧

### 快速上手
```bash
# 1. 更新 Cookie（10秒）
浏览器插件 → 复制 → 粘贴到 cookie.txt

# 2. 测试单公司
python test_single.py

# 3. 批量爬取
python main.py → 选择 2 → 选择文件类型
```

### 选择器调整
如果网站改版：
1. F12 查看元素
2. 复制 selector
3. 添加到 `selectors.json`
4. 重新运行

### Cookie 失效
症状：无法获取 VIP 数据
解决：重新获取 Cookie
